College Event Management System (Java + MySQL)

A desktop-based College Event Management System built using Java Swing, JDBC, and MySQL.
This project allows students to register, login, and manage college events efficiently.

🚀 Features
🔐 User Registration (MySQL storage)
🔑 User Login Authentication
🎯 Event Management (Add / View / Delete events)
👨‍🎓 Student Registration for Events
🖥️ Java Swing-based GUI
🗄️ MySQL Database Integration
⚙️ DAO-based layered architecture
🛠️ Technologies Used
Java (JDK 21+)
Java Swing (GUI)
JDBC (Database connectivity)
MySQL
OOP Concepts
DAO Design Pattern
📁 Project Structure
src/
│
├── ui/          # All GUI screens (Login, Registration, Dashboard)
├── dao/         # Database logic (UserDAO, EventDAO)
├── model/       # Entity classes (Student, Event, Registration)
├── util/        # DBConnection class
├── lib/         # MySQL JDBC driver
